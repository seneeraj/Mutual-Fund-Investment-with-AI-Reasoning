# 🧠 Mutual Fund Investment with AI Reasoning 

An intelligent, AI-powered investment simulation platform that combines **machine learning, financial reasoning, and user control** to optimize portfolio allocation, withdrawals, and long-term sustainability.

---

## 🚀 Key Features

### 📡 Market Intelligence

* Real-time market data using **NIFTY index**
* Technical indicators:

  * Moving Averages (MA50, MA200)
  * RSI (Relative Strength Index)
  * Volatility
* Machine Learning-based **Bull / Bear prediction**

---

### 🤖 AI Allocation Engine

* Dynamic portfolio allocation based on:

  * Risk profile (Conservative / Balanced / Aggressive)
  * Market condition (Bull / Bear)
* Adjusts exposure between:

  * Corporate (Debt)
  * Balanced
  * Flexi-cap
  * Index
  * Mid & Small cap

---

### ⚙️ Manual Override (Human + AI)

* Users can override AI allocation
* Real-time comparison:

  * AI vs User allocation
* Full transparency and flexibility

---

### ⚖️ Risk Analysis

* Portfolio **Risk Meter** (Low / Medium / High)
* Equity exposure calculation
* AI-based deviation warning:

  * Detects over-risking or under-investment

---

### 🧠 AI Reasoning Engine

* Explains decisions like a human advisor:

  * Trend analysis (MA50 vs MA200)
  * Momentum (RSI)
  * Risk (Volatility)
* Converts black-box ML into explainable AI

---

### 💰 Investment Simulation

* Monthly portfolio growth simulation
* Withdrawal strategy:

  * Priority-based fund withdrawal
* Yearly compounding and rebalancing

---

### 📊 Financial Planning Insights

* Safe withdrawal estimation
* Fund depletion tracking:

  * Which fund exhausts first
* Portfolio sustainability analysis

---

### 📦 Clean Dashboard UI

* Streamlit-based interactive dashboard
* Charts:

  * Market trend
  * Fund growth
* Clean portfolio summary
* Monthly tracking table

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Data:** yFinance (NIFTY index)
* **ML Model:** Random Forest (Scikit-learn)
* **Data Processing:** Pandas, NumPy

## 📈 How It Works

1. Fetches historical market data
2. Calculates technical indicators
3. Trains ML model to detect market trends
4. Generates AI-based allocation
5. Allows manual override
6. Simulates portfolio over time
7. Provides risk + withdrawal insights
